# Spring Boot Session Authentication

A simple Spring Boot MVC project demonstrating session-based authentication with user login, logout, and remember-me functionality using cookies.

## Features

- User registration and login
- Session-based authentication
- Remember-me functionality via cookies
- Logout functionality
- Thymeleaf templates

## Technologies

- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Oracle
- Jakarta Servlet API
